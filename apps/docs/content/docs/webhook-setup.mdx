# Webhook Setup

Owostack receives webhooks from Paystack on your behalf. This allows us to keep your subscription and usage data in sync automatically.

## 1. Find Your Webhook URL

Your webhook URL is:

```
https://api.owostack.dev/webhooks/{your-organization-id}
```

Find your organization ID in the Owostack dashboard URL:

- Dashboard URL: `https://dashboard.owostack.dev/app/abc123def`
- Webhook URL: `https://api.owostack.dev/webhooks/abc123def`

## 2. Configure Paystack

1. Go to [Paystack Dashboard](https://dashboard.paystack.com)
2. Navigate to **Settings** → **API Keys & Webhooks**
3. Under **Webhook URL**, paste your Owostack webhook URL
4. Click **Save**

<Callout type="important">
  Copy the **Webhook Secret** shown on the Paystack page. You'll need this in
  the next step.
</Callout>

## 3. Save Webhook Secret in Owostack

1. Go to your organization's **Settings** in the Owostack dashboard
2. Under **Webhooks**, paste the webhook secret from Paystack
3. Click **Save**

This secret is used to verify that incoming webhooks are actually from Paystack.

## 4. Test the Connection

Create a test subscription in Paystack:

1. Use Paystack's test mode
2. Create a new subscription using [Paystack's inline checkout](https://paystack.com/docs/payments/accept-payments/)
3. Complete the payment with a test card

If everything is configured correctly, you'll see:

- The subscription appear in your Owostack dashboard
- An event logged in the **Events** tab

## Events We Handle

| Event                    | What We Do                                |
| ------------------------ | ----------------------------------------- |
| `subscription.create`    | Create subscription, grant entitlements   |
| `subscription.enable`    | Activate subscription                     |
| `subscription.disable`   | Cancel/pause subscription                 |
| `subscription.not_renew` | Mark as pending cancellation              |
| `charge.success`         | Record payment, add credits if applicable |
| `invoice.payment_failed` | Mark subscription as past due             |

## Troubleshooting

### Webhooks not arriving

1. Verify the URL is correct in Paystack
2. Check that your organization ID matches
3. Ensure Paystack isn't in maintenance mode

### Invalid signature errors

1. Confirm the webhook secret matches exactly
2. Make sure there are no extra spaces when copying
3. Regenerate the secret in both Paystack and Owostack

### Events not processing

1. Check the **Events** tab for error logs
2. Verify your plans are configured correctly
3. Ensure customer emails match between systems
