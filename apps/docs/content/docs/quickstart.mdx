# Quickstart

Get to a working subscription flow end-to-end. This tutorial assumes you already have a payment provider account.

## 1. Create an organization

Sign up at [dashboard.owostack.dev](https://dashboard.owostack.dev) and create an organization.

## 2. Connect a provider account

In **Settings → Payment Providers**, add your provider credentials.

<Callout type="info">
  We validate your keys before saving. Invalid keys will be rejected.
</Callout>

## 3. Configure webhooks

In your provider dashboard, set the webhook URL to:

```
https://api.owostack.dev/webhooks/{your-organization-id}/{provider-id}
```

Example:

```
https://api.owostack.dev/webhooks/org_123/{provider-id}
```

<Callout type="warning">
  Webhooks are required for subscription state and entitlements. Without them,
  `check()` and `track()` won’t reflect real-time payment events.
</Callout>

## 4. Create a plan

In the dashboard:

1. Go to **Plans**
2. Click **Create Plan**
3. Set name, price, billing interval, and plan group
4. Attach features to the plan

## 5. Generate an API key

1. Go to **API Keys**
2. Click **Create New Key**
3. Copy the key (shown only once)

## 6. Install the SDK

```bash
npm install @owostack/core
# or
pnpm add @owostack/core
```

## 7. Use the SDK

```ts
import { Owostack } from "@owostack/core";

const owo = new Owostack({
  apiKey: process.env.OWOSTACK_API_KEY,
});

// Subscribe a customer
await owo.attach({
  customer: "user@example.com",
  plan: "pro-monthly",
});

// Gate a feature
const { allowed } = await owo.check({
  customer: "user@example.com",
  feature: "premium_features",
});

if (!allowed) {
  return res.status(403).json({ error: "Upgrade to Pro" });
}

// Track usage (metered billing)
await owo.track({
  customer: "user@example.com",
  feature: "api_calls",
  amount: 1,
});
```
