# Owostack

**Make Paystack billing as easy as Stripe Billing** - metered usage, entitlements, and subscriptions without the complexity.

## What is Owostack?

Owostack is billing infrastructure for Nigerian apps.

You connect your provider account (Paystack today), model plans and features in a dashboard, and then use a tiny SDK to enforce access and track usage.

## How it works

Model your pricing in Owostack

Handle payments

```ts
checkout
```

Check permissions and limits

```ts
check
```

Track usage

```ts
track
```

## Why use Owostack?

- Syncing subscription state (active, overdue payments, canceling) across many webhook edge cases
- Plan switching (upgrades, scheduled downgrades, add-ons)
- Usage limits (monthly recurring limits, one-time limits, credits)
- A simple, consistent interface (`attach`, `check`, `track`) regardless of provider details

## What Owostack is not

- A payment provider: you bring your own provider account and are never locked in to Owostack
- A full invoicing suite: Owostack focuses on subscription state, entitlements, and usage

## The Problem

Building billing with Paystack requires you to:

- Manage webhook endpoints and signature verification
- Track subscription states and handle edge cases
- Build usage metering and entitlement logic from scratch
- Handle failed payments, trials, and plan changes

## The Solution

Owostack handles all of this. You:

1. Link your Paystack account
2. Use our 3-method SDK
3. We manage everything else

```ts
import { Owostack } from "@owostack/core";

const owo = new Owostack({ apiKey: "owo_sk_..." });

// Subscribe a customer to a plan
await owo.attach({ customer: "user@example.com", plan: "pro" });

// Check what features they have access to
const { allowed, remaining } = await owo.check({
  customer: "user@example.com",
  feature: "api_calls",
});

// Track usage
await owo.track({
  customer: "user@example.com",
  feature: "api_calls",
  amount: 1,
});
```

## How It Works

```
┌─────────────┐                    ┌─────────────────────┐
│  Paystack   │ ──── webhooks ───► │   Owostack API      │
│  (your acc) │                    │  (handles events)   │
└─────────────┘                    └──────────┬──────────┘
                                              │
                                              ▼
                                   ┌─────────────────────┐
                                   │   Your App + SDK    │
                                   │  attach / check /   │
                                   │       track         │
                                   └─────────────────────┘
```

You point Paystack webhooks to us → we keep your data in sync → you query our API.
